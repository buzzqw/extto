Gap Filling — Fallback eD2k (cerca su eD2k se Jackett non trova nulla)

Se Jackett non trova risultati validi per un episodio mancante, EXTTO lo cerca sulla rete eD2k tramite aMule. Richiede aMule abilitato e connesso. Non richiede amuled fermo.

amuled deve essere fermo prima di salvare le impostazioni EC, porte e cartelle. Le modifiche a amule.conf vengono sovrascritte da amuled al riavvio. Ferma il servizio prima: sudo systemctl stop amule-daemon

Jellyfin / Plex

Jellyfin — Library Refresh —

Dopo ogni rinomina/spostamento file, EXTTO chiama /Library/Refresh per aggiornare subito la libreria senza aspettare la scansione schedulata.
Ottieni la API Key da Pannello di controllo → Chiavi API.

Plex — Library Refresh —

Dopo ogni rinomina/spostamento, EXTTO chiama /library/sections/all/refresh per aggiornare subito tutte le librerie Plex.
Ottieni il token da Account → Preferenze XML → token oppure via questa guida.

FlareSolverr — Bypass Cloudflare —

FlareSolverr usa Chrome headless per superare le protezioni Cloudflare. Richiesto per il scraping di ext.to.
Installa con Docker: docker run -d --name flaresolverr -p 8191:8191 ghcr.io/flaresolverr/flaresolverr:latest

Lascia vuoto per disabilitare. Senza FlareSolverr ext.to potrebbe restituire 403.

Svuota Archivio Jackett/Scraping

Rimuove dall'archivio i magnet link trovati da Jackett o dai siti vecchi di un tot di giorni. Utile se l'archivio ha superato le decine di migliaia di righe rallentando il sistema.

Filtra per Parola Chiave

Cerca nell'archivio per titolo ed elimina tutti i record corrispondenti. Più parole separate da virgola o spazio applicano un OR (basta una corrispondenza).

Pulizia Cestino

Elimina manualmente i file nel cestino (—) più vecchi del numero di giorni configurato in Impostazioni → Giorni Retention Cestino.
La pulizia avviene anche automaticamente ad ogni ciclo del motore se il parametro è configurato.

aMule / ed2k Stato sconosciuto
Configura in aMule / ed2k → Impostazioni.
aria2c Service Stato sconosciuto
Gestisci il demone aria2c configurato in Configurazione → Client.
Avvia avvia il processo aria2c con RPC abilitato usando il path e il secret configurati. Il Watchdog di EXTTO monitorerà i download e avvierà il post-processing al completamento.
Gestione Servizio (systemd)
Riavvia Motore EXTTO Termina il servizio Extto e lo riavvia dopo circa 10 secondi... non temere! Utile per forzare lo svuotamento della memoria cache o ricaricare in modo "pulito" i file di configurazione se modificati a mano.
